# Video-From-PowerPoint-using-Gemini

This Google Colab (JuPyter script) will take you step by step from a PowerPoint
all the way to a working MP4 file. 

Step 1  - Google Drive Structure 
Step 2 - Mount Drive in Colab
Step 3 - Set up requirements (API Key, etc)
Step 4 - Generate Narrations using AI
Step 5 - Generate audio (wav) files using Google TTS
Step 6 - Embed wav files into PowerPoint
Step 7 - Export final as MP4

Documentation of each step is provided in the JuPyter notebook.
