# Video-From-PowerPoint-using-Gemini

This Google Colab (JuPyter script) will take you step by step from a PowerPoint
all the way to a working MP4 file. 

Step 1  - Google Drive Structure <br/>
Step 2 - Mount Drive in Colab<br/>
Step 3 - Set up requirements (API Key, etc)<br/>
Step 4 - Generate Narrations using AI<br/>
Step 5 - Generate audio (wav) files using Google TTS<br/>
Step 6 - Embed wav files into PowerPoint<br/>
Step 7 - Export final as MP4<br/><br/>

Documentation of each step is provided in the JuPyter notebook.
